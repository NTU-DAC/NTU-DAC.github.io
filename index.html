Website Tutorial

Learning Process

Click Schedule to view the course timeline, then select the Lecture Video you want to study.
After learning, click Download to get the assignment and test files, then test your code locally to check correctness.
Once tested, click the Assignment Submission link for that week's course and upload your Python code (only .py files are allowed).

Link Descriptions

Lecture Video: Link to the course video.
Colab Notebook: Link to the course materials.
CAssignment Download: Downloadable files including assignment instructions, test code, and test data.
Assignment Submission: Link for submitting assignments.

Local Testing

Click Download to get the zip file assignment.zip .
Rename your code file to solution.py and place it in the same directory as grader.py :
assignment/
├── grader.py
├── solution.py
└── tests/
    ├── input1.txt
    ├── output1.txt
    ├── input2.txt
    ├── output2.txt
Navigate to the folder where grader.py is located:
cd ~/Desktop/assignment
Run the following command:
python grader.py

About Python Arena

Welcome to Python Arena! This program is established by Academic Department of the 7th NTU DAC to strengthen our members' Python skills.

The course spans several weeks, combining video lessons with practice assignments to help you consolidate your foundational knowledge of Python.

By the end of the course, we hope that everyone will have developed sufficient Python proficiency to participate in future corporate projects.

Special thanks to Bing-Chen (Benson) Chiu from NTU DAC 6th for creating a significant portion of the course handouts and lecture videos.
